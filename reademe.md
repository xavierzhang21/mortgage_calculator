# 💰 Personal Accounting Tool (个人记账本)

A simple and elegant browser-based accounting tool.

## ✨ Features
- Add income & expense records
- Category tracking
- Monthly trend chart
- Expense breakdown (pie chart)
- Local storage (no backend needed)
- Export to CSV

## 🚀 Demo
You can deploy easily using GitHub Pages.

## 📦 How to Use
1. Clone the repo
2. Open `index.html` in browser

## 🌐 Deployment (GitHub Pages)
1. Go to repo Settings
2. Enable Pages
3. Select `main` branch / root
4. Done 🎉

## 🛠 Tech Stack
- HTML
- CSS
- Vanilla JavaScript
- Chart.js

## 📁 Data Storage
Uses browser `localStorage` (key: `records_v2`)
